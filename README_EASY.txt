# 전구체 대시보드 초간단 설치 가이드

이 프로그램은 복잡한 설정 없이 바로 사용할 수 있도록 만들어졌습니다.

### 윈도우(Windows) 사용자
1. 이 폴더 안에 있는 **`setup_and_run.bat`** 파일을 마우스로 두 번 클릭하세요.
2. 검은색 창이 뜨면서 자동으로 필요한 것들을 설치합니다. (처음 한 번만 시간이 좀 걸립니다.)
3. 설치가 끝나면 인터넷 창이 열리면서 대시보드가 나타납니다.

### 주의사항
* 컴퓨터에 **Node.js**라는 프로그램이 깔려 있어야 합니다. 
  (안 깔려 있다면 https://nodejs.org/ 에서 왼쪽의 'LTS' 버튼을 눌러 설치하세요.)
* 검은색 창을 끄면 프로그램도 종료됩니다. 사용 중에는 창을 닫지 마세요.

### 팁: 바탕화면 앱으로 쓰기
브라우저 주소창 옆의 '설치' 버튼을 누르면 바탕화면에 아이콘이 생겨서 진짜 프로그램처럼 쓸 수 있습니다.
